### When Creating Shopping Lists, Always:

**STEP 1: WEB SEARCH (MANDATORY)**
```
1. Search: "iHeartPublix Publix ad [current date]"
2. Fetch: Complete BOGO list from iHeartPublix
3. Search: "Southern Savers Publix [current date]"
4. Verify: Sale dates and current day
5. Note: Days remaining, store hours, timing
```

**STEP 2: EXTRACT PRICING DATA**
```
For each item needed:
- Find exact price from current flyer
- Note if BOGO (format: "BOGO $X.XX")
- Check purchase limits
- Find digital coupon amount (format: "$X/X")
- Find Ibotta offer amount (format: "$X/X")
- Never use "approximately" for verifiable prices
```

**STEP 3: CREATE TIER SYSTEM**
```
Tier 1: MUST BUY (mandatory meals + breakfast + household)
Tier 2: BUY IF BOGO (flexible meals + stock-up items)
Tier 3: SKIP THIS WEEK (wait for better sales)
```

**STEP 4: CALCULATE COSTS**
```
Scenario 1: Minimal (Tier 1 only)
Scenario 2: Smart (Tier 1 + selective Tier 2)
Scenario 3: Maximum (all tiers)

Compare to budget: $240-300 weekly target
```

**STEP 5: FORMAT PROPERLY**
```
Required header elements:
- Sale dates (start - end)
- Source (iHeartPublix/Southern Savers)
- Sale status (days remaining/last day/first day)
- Budget target
- Store location

Required sections:
- Tier 1: Must Buy (with subtotal)
- Tier 2: Buy if BOGO (with subtotal)
- Tier 3: Skip This Week (with reason)
- Three cost scenarios
- Quick printable list
- Coupon loading checklist
- Critical reminders (48-hour Ibotta!)
```
